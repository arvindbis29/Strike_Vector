# Insights Generator UI (React + Vite + Tailwind) - Enhanced

## Features (Enhanced)
- JSON input editor with validation and localStorage
- Tabs: Input / Response / Raw JSON
- Export response as JSON
- Download insights as PDF (jsPDF)
- Dark/Light theme toggle
- Responsive layout and improved UI
- Error handling and loading states

## Setup

1. Install
```
npm install
```

2. Run dev server
```
npm run dev
```

3. API endpoint
By default the app posts to `http://localhost:8080/insights/generate/`. Modify `src/utils/api.js` if needed.
